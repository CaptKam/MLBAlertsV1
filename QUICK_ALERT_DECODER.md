# Quick Alert Decoder - At-a-Glance Guide

## 🚨 **INSTANT RECOGNITION**

### **Emojis = Alert Type**
- 🎮 = Game started (first pitch thrown)
- 💥 = Power hitter at bat  
- 🚀 = Home run happened OR elite power threat
- 💰 = ROI situation (power + scoring opportunity)
- 🔥 = High-scoring opportunity OR hot weather
- 🎯 = Bases loaded OR high-precision situation
- ⚡ = Runner on 3rd, easy scoring
- 💨 = Wind helping hitters
- 🌡️ = Temperature boosting offense
- 🏏 = Hit just happened
- ⚾ = Strikeout occurred
- 🏃 = Runner advancement
- ⚠️ = 7th inning (crunch time)
- 🤖 = AI providing extra insight

---

## 📊 **KEY NUMBERS TO WATCH**

### **Home Run Probabilities (P(HR)):**
```
6%+ = 🔥 RED HOT (bet it)
4-5% = 🟡 STRONG (good value) 
2-3% = 🟢 DECENT (situational)
<2% = ⚪ AVERAGE (no tier shown)
```

### **Tiers:**
```
Tier A = Elite (top 5% of situations)
Tier B = Strong (top 20% of situations)  
Tier C = Above average
No tier = Below average probability
```

### **Weather Boosts:**
```
+30%+ = 🔥 Major boost (smash the over)
+15-29% = 🟡 Good boost (consider over)
+5-14% = 🟢 Minor boost (slight edge)
-10%+ = ❄️ Weather hurting (under edge)
```

### **See This → Do This:**

**💥 + Tier A + 4%+ P(HR)** → Bet home run props
**💰 + Runners on base** → RBI props, run line  
**🔥 + 85%+ scoring** → Over inning total
**💨 + 20%+ boost** → Over game total

## 📱 **MOBILE QUICK SCAN**

**First Line = Situation**
**Second Line = Player/Details**  
**📊 Line = Math/Probability**
**💨 Line = Weather Impact**
**🤖 Line = AI Insight (if enabled)**

**Example:**
```
💥 POWER HITTER ALERT!          ← Situation
Judge at bat with 32 HRs!       ← Player
📊 P(HR): 4.2% | Tier: A        ← Math says elite
💨 Wind helping: +28% boost     ← Weather helping
```

**Translation:** Elite power hitter + weather help = bet the long ball
