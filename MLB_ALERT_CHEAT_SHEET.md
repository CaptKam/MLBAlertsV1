# MLB Alert System - Complete Cheat Sheet

## 🎯 How to Read Your Alerts

This guide explains every type of alert your system sends and what they mean for betting/viewing opportunities.

---

## 📱 **GAME STATUS ALERTS**

### 🎮 **Game Start Alert**
```
🎮 PLAY BALL!
First pitch thrown!
Game is now underway!
```
**What it means:** The first pitch has actually been thrown (not just scheduled)
**Why it matters:** Games are now live for in-play betting
**Action:** Check live odds, game has officially started

---

## ⚾ **SCORING OPPORTUNITY ALERTS**

### 💰 **High-Probability Scoring Situations**

#### 🔥 **Runners on 2nd & 3rd, No Outs** (~87% scoring chance)
```
🔥 SCORING OPPORTUNITY!
Runners on 2nd and 3rd with NO OUTS!
High probability of scoring this inning
```
**Translation:** Almost guaranteed run(s) coming
**Betting impact:** Over/under, run line, inning totals

#### 🎯 **Bases Loaded, No Outs** (~85% scoring chance)
```
🎯 BASES LOADED - NO OUTS!
Maximum pressure situation!
```
**Translation:** Multiple runs very likely
**Betting impact:** Big inning potential, over bets

#### ⚡ **Runner on 3rd, No Outs** (~75% scoring chance)
```
⚡ RUNNER ON THIRD - NO OUTS!
Score with almost any contact
```
**Translation:** Easy scoring position
**Betting impact:** At least 1 run very likely

## 🚀 **POWER HITTER ALERTS**

### 💥 **Basic Power Hitter Alert**
```
💥 POWER HITTER ALERT!
Aaron Judge at bat with 32 HRs this season!
📊 P(HR): 4.2% | Tier: A
💨 Wind helping: +28% boost
```
**How to read:**
- **Name + HR count:** Player's season power
- **P(HR): 4.2%:** Math-calculated home run probability this at-bat
- **Tier A/B/C:** Power level (A = Elite, B = Strong, C = Above average)
- **Wind boost:** Weather helping or hurting (+% = helping, -% = hurting)

### 🚀 **High-Confidence Power Alert**
```
🚀 HIGH-CONFIDENCE POWER ALERT!
Juan Soto (28 HRs) - Elite power threat!
📊 P(HR): 5.1% | Tier: A
```
**Translation:** Elite power hitter (Tier A only) with very high HR probability
**Action:** Consider home run props, over bets

### 💰 **ROI Alert: Power + Scoring**
```
💰 ROI ALERT: POWER + SCORING!
Mookie Betts (25 HRs) at bat with runners on base!
📊 P(HR): 3.8% | Tier: B
```
**Translation:** Strong power hitter + runners = multiple RBI potential
**Action:** RBI props, run line, big inning bets

## 🎯 **ALERT PRIORITY SYSTEM**

### **Tier A Alerts** (Highest Priority)
- Elite power hitters (4%+ HR probability)
- Bases loaded situations
- Weather-boosted conditions

### **Tier B Alerts** (High Priority)  
- Strong power hitters (2.5%+ HR probability)
- RISP situations
- Tie games late

### **Tier C Alerts** (Medium Priority)
- Above-average situations (1.5%+ HR probability)
- Inning changes
- General scoring opportunities

---

## 💡 **QUICK REFERENCE**

### **Probability Meanings:**
- **P(HR): 5%+** = Elite situation, bet the long ball
- **P(HR): 3-4%** = Strong situation, good value
- **P(HR): 2-3%** = Above average, situational
- **P(HR): <2%** = Below average (alert won't show tier)

### **Weather Impact:**
- **+25% boost** = Significant help for hitters
- **+10% boost** = Moderate help
- **-10% penalty** = Weather hurting offense

### **Base Situation Rankings (Scoring %):**
1. **Bases Loaded, 0 outs:** ~85%
2. **2nd & 3rd, 0 outs:** ~87%  
3. **3rd base, 0 outs:** ~75%
4. **2nd base, 0 outs:** ~63%
5. **RISP, 1 out:** ~45-55%

## 🎯 **ACTION GUIDE**

**When you see Tier A alerts:** Strong betting opportunity
**When you see weather boosts 20%+:** Over bets, HR props  
**When you see RISP + Power combo:** RBI props, run lines
**When you see 9th inning ties:** Extra innings, walk-off props
**When you see multiple alerts same game:** Stack your bets

Your alerts are mathematically calculated for maximum accuracy and timing!
